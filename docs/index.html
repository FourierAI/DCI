The problem

More labels.
Less signal.

MLLM recognition accuracy can collapse as the candidate label space expands. We call this Performance Collapse in Long Sequence Recognition.

01

Entropy escalates

Every added candidate increases uncertainty and makes the global decision harder.

02

Attention dilutes

Relevant visual-label interactions receive progressively weaker attention weights.

03

SNR decays

The correct category becomes harder to recover from increasingly noisy contexts.

Our approach

Think locally.
Scale globally.

DCI replaces one difficult global classification with a sequence of smaller local decisions—without training, fine-tuning, or architectural modification.

01

Divide

Partition the candidate set into compact groups of size K.

02

Conquer

Ask the same MLLM to solve independent local recognition tasks.

03

Prune

Discard rejected branches and retain only plausible candidates.

04

Refine

Repeat until the search converges to one final prediction.

Control context length.
Compress the search.

Abstract

Divide the search space.
Preserve the signal.

Multimodal Large Language Models have demonstrated strong capabilities across a wide range of vision-language tasks. However, when applied to large-scale image classification, their performance degrades significantly as the label space expands—a phenomenon we define as Performance Collapse in Long Sequence Recognition.

Through an information-theoretic analysis, we reveal that this collapse stems from a fundamental conflict between escalating information entropy and attention dilution and decay. To mitigate this, we propose Divide-and-Conquer Inference, a training-free strategy that recursively decomposes complex global classification into simpler localized sub-problems and dynamically prunes the search space.

Extensive experiments on ImageNet-1K and ImageNet-21K demonstrate consistent accuracy improvements, enabling lightweight open-source models to rival or surpass much larger frontier models without additional training or fine-tuning.

Open source

Try DCI in
three commands.

A unified, dataset-agnostic implementation for any multimodal model served through an OpenAI-compatible API.

quickstart.sh
# Install DCI
git clone https://github.com/FourierAI/DCI.git
cd DCI && pip install -e .

# Run divide-and-conquer inference
dci-eval \
  --dataset imagenet1k \
  --model Qwen/Qwen3-VL-2B-Instruct \
  --image-root /path/to/imagenet/val \
  --k-values 100 50 20 10
